# AxeKAS — Umbrel App Store

Custom Umbrel community app store for **AxeKAS**: a self-hosted Kaspa full-node dashboard for umbrelOS.

## Add this store to Umbrel

In umbrelOS → App Store → **Community App Stores**, paste:

```
https://github.com/AkaCurtis/kaspa-umbrel
```

Then install **AxeKAS** (app ID: `rvn-curtis-axekaspa`).

## What it includes

- Live Kaspa node sync status and block DAG stats
- Peer count, network difficulty, mempool size
- Solo pool stratum endpoint (port 4827)
- Worker hashrate charts and history
- Settings for payout address and vardiff
